# User Flow

## Deskripsi

User merupakan masyarakat umum yang mengakses website tanpa perlu login.

Tujuan utama user adalah mencari informasi pengepul berdasarkan jenis limbah yang dimiliki.

---

# User Journey

Home
↓
Pilih Kategori Limbah
↓
Membaca Edukasi Limbah
↓
Melihat Daftar Pengepul
↓
Melihat Detail Pengepul
↓
Menghubungi Pengepul melalui WhatsApp

---

# Flow Detail

## 1. Home Page

User membuka website.

Menampilkan:

* Hero Section
* Daftar kategori limbah
* Informasi singkat WasteLink
* Tombol eksplorasi kategori

Aksi:

* Klik kategori

↓

## 2. Halaman Detail Kategori

User melihat:

* Nama kategori
* Gambar kategori
* Deskripsi kategori
* Edukasi limbah
* Daftar pengepul terkait

Aksi:

* Memilih pengepul

↓

## 3. Halaman Detail Pengepul

User melihat:

* Nama pengepul
* Gambar pengepul
* Kategori yang diterima
* Alamat
* Nomor telepon
* WhatsApp
* Jam operasional
* Deskripsi

Aksi:

* Klik tombol WhatsApp

↓

## 4. Hubungi Pengepul

User diarahkan ke WhatsApp.

Flow selesai.
